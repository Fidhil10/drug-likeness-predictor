# Drug-likeness-predictor
An intelligent machine learning platform for predicting the drug-likeness of molecular compounds using molecular descriptors, molecular fingerprints, and predictive analytics. The application supports real-time inference, batch compound screening, interactive visualizations, REST API integration, and exportable reports for drug discovery and molecular screening workflows.

# Overview
An intelligent machine learning platform for predicting the drug-likeness of molecular compounds using molecular descriptors, molecular fingerprints, and predictive analytics. The application supports real-time inference, batch compound screening, interactive visualizations, REST API integration, and exportable reports for drug discovery and molecular screening workflows.

# Installation

# Clone Repository
git clone https://github.com/Fidhil10/drug-likeness-predictor.git
cd drug-likeness-predictor

# Create Virtual Environment
python -m venv venv

# Activate Environment

Windows:

venv\Scripts\activate

Linux/macOS:

source venv/bin/activate

# Install Dependencies
pip install -r requirements.txt

# Run the Application
# Streamlit
streamlit run app.py
# FastAPI
uvicorn main:app --reload

# How It Works
